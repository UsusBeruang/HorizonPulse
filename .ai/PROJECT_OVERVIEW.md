# Horizon Pulse

Modern telemetry and automation companion platform for Forza Horizon 6.

## Core Features

- Real-time UDP telemetry communication
- FH6 telemetry packet parsing
- Modern dark-themed WPF dashboard
- Live telemetry inspector
- Lightweight automation using ViGEm
- Race and lap monitoring
- Runtime-safe automation behavior

## Technology Stack

- .NET 8
- WPF
- MVVM
- Serilog
- Generic Host
- ViGEm

## Goals

- Stable telemetry parsing
- Clean and readable dashboard UI
- Lightweight automation assistance
- Maintainable architecture
- Fast iteration and development

## Non-Goals

The project intentionally avoids:
- Full autonomous driving
- Machine learning systems
- Replay systems
- Computer vision systems
- Over-engineered architecture
