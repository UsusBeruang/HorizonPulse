# Horizon Pulse - Forza Telemetry Dashboard

A casual, real-time telemetry dashboard for Forza Motorsport that displays vehicle data from UDP packets.

## Features

- Real-time vehicle telemetry display
- Speed, RPM, gear, lap information
- Throttle and brake inputs
- Race timing data
- Connection status indicator

## Getting Started

1. Run Forza Motorsport with UDP telemetry enabled
2. Start this application
3. View real-time telemetry data

## Requirements

- .NET Framework 4.7.2 or later
- Forza Motorsport with UDP telemetry enabled
